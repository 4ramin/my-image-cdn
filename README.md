# my-image-cdn
This code will help you to create your custom CDN Edge.

## Installation

### Apache
- Edit `config.php`
- Edit `.htaccess`

### NGINX
- Edit `config.php`
- Go (https://blog.lael.be/demo-generator/imagecdn/my-example-site.com.php) and follow the instructions.


## 설치 방법
Apache 환경에서 사용하는 것을 권장합니다.  
가끔씩 이 코드의 구조를 개선합니다. 이미 이 코드를 적용하였고, 잘 동작하고 있다면, 굳이 최신버전으로 패치할 필요는 없습니다.
